# AGENTS.md

## Правила проекта

- Сначала строим MVP, а не production-платформу.
- Предпочитаем простой, модульный Python-код без тяжёлых абстракций.
- Вся логика внешних API должна жить в отдельных client-модулях.
- Все scoring-формулы должны лежать в чистых domain-модулях.
- Везде, где разумно, использовать type hints.
- Использовать стиль SQLAlchemy 2.x.
- Добавить pytest-тесты для core formulas.
- Все пороги и коэффициенты хранить в settings, а не хардкодить в логике.
- Все assumptions и TODO нужно явно документировать.
- После каждой фазы реализации кратко описывать прогресс и то, что осталось сделать.
